Built a single paged web app that is based on native Node.Js and javascript capabilities to allow users to open a chord pro file hosted by a Node.js server. Within the browser you can drag chords and words around and reupload the updated file in chord pro format 


 
Version: Node Version 6.11.3,coded on Windows 
Install: No NPM Module_name
Launch: node server.js
Testing:localhost:3000/assignment2.html,
